# auditoria-interna
A simple PyQt5 UI to make everyday easier at GBS.

Auditoria Interna v1.06

----------------------------

To use the application,

1. Install the requirements using cmd/conda console.
	pip install -r requirements.txt
2. Run main.py file.
3. Navigate through the GUI.
